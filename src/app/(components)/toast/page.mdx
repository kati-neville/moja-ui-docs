# Toast

Toasts are non-interactive, passive, and asynchronous short messages for users.

## Installation

<CodeGroup>

```bash {{ title: 'npm' }}
npm install @mojaui/toast
```

```bash {{ title: 'yarn' }}
yarn add @mojaui/toast
```

```bash {{ title: 'pnpm' }}
pnpm install @mojaui/toast
```

```bash {{ title: 'bun' }}
bun add @mojaui/toast
```

</CodeGroup>

## Variants

In Moja UI, the Toast component has 4 main variants, which can be defined by the <code>variant</code> prop on the root layer.

- Success Toast
- Error Toast
- Info Toast
- Warning Toast

## Usage

The Toast component can be used as follows:

```js
import { Toast } from '@mojaui/toast'

export function Index() {
  return (
    <Toast variant="success">
      <Toast.Icon></Toast.Icon>
      <Toast.Content>
        <Toast.Title></Toast.Title>
        <Toast.Description></Toast.Description>
      </Toast.Content>
      <Toast.Close></Toast.Close>
    </Toast>
  )
}
```
