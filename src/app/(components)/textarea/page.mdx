# Textarea

The Textarea component is a multi-line input component which allows you to write large texts.

## Installation

<CodeGroup>

```bash {{ title: 'npm' }}
npm install @mojaui/textarea
```

```bash {{ title: 'yarn' }}
yarn add @mojaui/textarea
```

```bash {{ title: 'pnpm' }}
pnpm install @mojaui/textarea
```

```bash {{ title: 'bun' }}
bun add @mojaui/textarea
```

</CodeGroup>

## Usage

The Input component can be used as follows:

```js
import { Textarea } from '@mojaui/textarea'

export function Index() {
  return (
    <Textarea.Root>
      <Textarea.Label></Textarea.Label>
      <Textarea />
    </Textarea.Root>
  )
}
```

## With Error Message

You can combine the `isInvalid` prop on the <b>`Root`</b> component and <b>`ErrorText`</b> to display the error message

```js
import { Textarea } from '@mojaui/textarea'

export function Index() {
  return (
    <Textarea.Root isInvalid={true}>
      <Textarea.Label></Textarea.Label>
      <Textarea />
      <Textarea.ErrorText></Textarea.ErrorText>
    </Textarea.Root>
  )
}
```

## Controlled Textarea

```js
import { Textarea } from '@mojaui/textarea'

export function Index() {
  const [value, setValue] = React.useState('')

  const handleValueChange = (event) => setValue(event.target.value)

  return (
    <Textarea.Root>
      <Textarea.Label>Your first message</Textarea.Label>
      <Textarea value={value} onChange={handleValueChange} />
    </Textarea.Root>
  )
}
```

## API Reference

### Root

<Properties>
  <Property name="isInvalid" type="boolean">
    Whether or not the textarea is invalid.
  </Property>
  <Property name="disabled" type="boolean">
    Whether or not the textarea is disabled.
  </Property>
  <Property name="required" type="boolean">
    Whether or not the textarea field is required.
  </Property>

  <Property name="[data-disabled]">
    This is present when the textarea is disabled
  </Property>
  <Property name="[data-hover]">
    This is present when the textarea is hovered.
  </Property>
  <Property name="[data-focus]">
    This is present when the textarea is focused.
  </Property>
</Properties>

### Textarea

The Textarea component forwards all the props/attributes from the base <a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/textarea#attributes" target="_blank">HTML Textarea element</a>.
