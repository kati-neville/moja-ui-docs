# Link

The Link component which is an extension of the anchor element allows for easy customization.

## Installation

<CodeGroup>

```bash {{ title: 'npm' }}
npm install @mojaui/link
```

```bash {{ title: 'yarn' }}
yarn add @mojaui/link
```

```bash {{ title: 'pnpm' }}
pnpm install @mojaui/link
```

```bash {{ title: 'bun' }}
bun add @mojaui/link
```

</CodeGroup>

## Variants

In Moja UI, we have 2 types of Link displays, which can be defined using the <code>display</code> prop on the Button component

- Compact Link
- Fluid Link

In Moja UI, the Link component has 4 main variants, which can be defined by the <code>variant</code> prop.

- Default Link
- Primary Chip
- Light Chip
- Danger Chip

## Usage

The Link component can be used as follows:

```js
import { Link } from '@mojaui/link'

export function Index() {
  return (
    <Link variant="primary" display="compact">
      Link Text
    </Link>
  )
}
```
