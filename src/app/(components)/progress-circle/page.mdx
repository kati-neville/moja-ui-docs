# Progress - Circular

This component is used to indicate an undetermined wait period or visually represent the duration of a process.

## Installation

<CodeGroup>

```bash {{ title: 'npm' }}
npm install @mojaui/progress
```

```bash {{ title: 'yarn' }}
yarn add @mojaui/progress
```

```bash {{ title: 'pnpm' }}
pnpm install @mojaui/progress
```

```bash {{ title: 'bun' }}
bun add @mojaui/progress
```

</CodeGroup>

## Usage

The progress component can be used as follows:

```js
import { Progress } from '@mojaui/progress'

export function Index() {
  return (
    <Progress.Root>
      <Progress.Label></Progress.Label>
      <Progress.Value></Progress.Value>
      <Progress.Circle>
        <Progress.CircleTrack />
        <Progress.CircleRange />
      </Progress.Circle>
    </Progress.Root>
  )
}
```

## Determinate and Indeterminate state

- The <b>Determinate</b> state displays how long an operation will take.
- The <b>Indeterminate</b> state visualizes and unspecified wait time.

These states can be defined using the <code>variant</code> prop.

```js
import { Progress } from '@mojaui/progress'

export function Indeterminate() {
  return (
    <Progress.Root variant="indeterminate">
      <Progress.Label></Progress.Label>
      <Progress.Circle>
        <Progress.CircleTrack />
        <Progress.CircleRange />
      </Progress.Circle>
    </Progress.Root>
  )
}
```

## API Reference

<Properties>

<Property name="variant" type="indeterminate | determinate"></Property>
<Property name="value" type="number" defaultValue="0">
  The current value of the progress bar.
</Property>
<Property name="onValueChange" type="(value: number) => void">
  A function that runs whenever the progress value changes.
</Property>
<Property name="min" type="number" defaultValue="0">
  The minimum allowed value of the progress bar.
</Property>
<Property name="max" type="number" defaultValue="100">
  The maximum allowed value of the progress bar.
</Property>
</Properties>
