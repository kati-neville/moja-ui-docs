# Avatar

An Avatar component can be used as a visual identifier on your website, often used in user profile sections.

## Installation

<CodeGroup>

```bash {{ title: 'npm' }}
npm install @mojaui/avatar
```

```bash {{ title: 'yarn' }}
yarn add @mojaui/avatar
```

```bash {{ title: 'pnpm' }}
pnpm install @mojaui/avatar
```

```bash {{ title: 'bun' }}
bun add @mojaui/avatar
```

</CodeGroup>

## Usage

The Avatar component can be used as follows:

```js
import { Avatar } from '@mojaui/avatar'

export function Index() {
  return (
    <Avatar>
      <Avatar.Fallback></Avatar.Fallback>
      <Avatar.Image></Avatar.Image>
      <Avatar.Decorator></Avatar.Decorator>
    </Avatar>
  )
}
```
