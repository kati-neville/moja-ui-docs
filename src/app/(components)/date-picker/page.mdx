# Date Picker

A component that allows users select a date from a calendar.

The Date Picker component has been built upon the <a href="https://date-fns.org/">date fns</a> library.

## Installation

<CodeGroup>

```bash {{ title: 'npm' }}
npm install @mojaui/date-picker
```

```bash {{ title: 'yarn' }}
yarn add @mojaui/date-picker
```

```bash {{ title: 'pnpm' }}
pnpm install @mojaui/date-picker
```

```bash {{ title: 'bun' }}
bun add @mojaui/date-picker
```

</CodeGroup>
