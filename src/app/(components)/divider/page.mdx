# Divider

The Divider component creates a thin line that helps organize elements and enhance visual hierarchy. It's useful for visually separating content in lists or groups.

## Installation

<CodeGroup>

```bash {{ title: 'npm' }}
npm install @mojaui/divider
```

```bash {{ title: 'yarn' }}
yarn add @mojaui/divider
```

```bash {{ title: 'pnpm' }}
pnpm install @mojaui/divider
```

```bash {{ title: 'bun' }}
bun add @mojaui/divider
```

</CodeGroup>

## Usage

The Divider component can be used as follows:

```js
import { Divider } from '@mojaui/divider'

export function Index() {
  return <Divider />
}
```
