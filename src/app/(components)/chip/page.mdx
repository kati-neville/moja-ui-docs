# Chip

Chips are compact elements that represent an input, attribute, or action.

## Installation

<CodeGroup>

```bash {{ title: 'npm' }}
npm install @mojaui/chip
```

```bash {{ title: 'yarn' }}
yarn add @mojaui/chip
```

```bash {{ title: 'pnpm' }}
pnpm install @mojaui/chip
```

```bash {{ title: 'bun' }}
bun add @mojaui/chip
```

</CodeGroup>

## Usage

```js
import { Chip } from '@mojaui/chip'
import { CloseIcon } from 'moja-icons'

export function Index() {
  return (
    <Chip.Root>
      <Chip.Label>A</Chip.Label>
      <Chip.Icon>
        <CloseIcon />
      </Chip.Icon>
    </Chip.Root>
  )
}
```

## Variants

In Moja UI, the Chip component has 2 main variants, which can be defined by the <code>variant</code> prop on the root layer.

- Gray Chip
- Primary Chip

And 2 types which can be defined by the <code>type</code> prop on the root layer.

- Basic Chip
- Outlined Chip

```js
import { Chip } from '@mojaui/chip'
import { CloseIcon } from 'moja-icons'

export function Index() {
  return (
    <Chip.Root variant="primary" type="outline">
      <Chip.Label>Sample Label</Chip.Label>
      <Chip.Icon>
        <CloseIcon />
      </Chip.Icon>
    </Chip.Root>
  )
}
```

## Accessibility

### Keyboard Support

<Properties>
  <Property name="Tab">
    Moves focus to either the pressed item or the first item in the group.
  </Property>
  <Property name="Space | Enter">Selects/Deselects the item.</Property>
  <Property name="Shift + Tab">
    Moves focus to the previous focusable element
  </Property>
